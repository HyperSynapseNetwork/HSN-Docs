# HSN 交流グループルール

> グループ内の良好な交流雰囲気を維持し、メンバーの安全と体験を保障するため、全てのメンバーは以下のグループルールをよく読み、遵守してください。入群をもって本ルールに同意したものとみなします。

## 一、身分と安全

1. **身分確認の原則**：女性を自称する全てのユーザーは、自ら進んで身分証の性別情報を公開して確認しない限り、全て「男の娘」として扱います。グループメンバーは注意を高め、ネット詐欺に警戒してください。
2. **違法行為の禁止**：個人情報の特定（開盒）、個人情報の探求（人肉検索）、プライバシーの販売、わいせつ物の拡散などの違法行為は一切禁止します。発見次第、即座にグループから退会処分とします。重大な結果を生じた場合、管理グループは警察への通報および法的手段による責任追及の権利を留保します。
3. **メンバーとBotへの通報禁止**：本グループのメンバーおよび HSNBot に対する悪意のある通報を禁止します（わいせつ物の拡散を行う者を除く）。

## 二、発言とコンテンツ規範

1. **センシティブな話題の禁止**：本グループでは宗教、政治に関する内容の議論を許可しません。また、関連するリンク、動画、記事などのメディア素材の送信も禁止します。歴史的人物に関するもので、Bilibili などの主流プラットフォームで通常公開可能な画像・動画はこの限りではありません。
2. **政治的立場の表明禁止**：本グループは一切の政治的立場や傾向を持たず、グループの名義で政治的見解を表明することを禁止します。
3. **グレーマーケット（闇市場）宣伝の管理**：許可のないグレーマーケットの宣伝（論文代筆、ギャンブルサイトへの誘導などを含むがこれらに限らない）は、初回警告、2回目は1日のミュート、3回目は永久追放とします。
4. **コンテンツの尺度管理**：
   - ギャンブルと薬物は拒否し、あらゆる薬物やギャンブル関連コンテンツを断固として拒否します
   - 性的コンテンツは尺度に注意し、過度に露骨なものは30秒以内に自らグループアルバムに移動してメンバーの閲覧に供すること。適切に処理されなかった場合は1時間以上のミュートとします
   - 実写の過激な性的コンテンツは、猶予期間なしで直接ミュートとします

## 三、交流秩序

1. **紛争処理**：グループ内で口論が発生した場合、双方は冷静かつ理性的にコミュニケーションを取るか、速やかに管理者に連絡して介入を要請してください。個人攻撃や挑発、罵り合いは禁止します。
2. **スパム対策**：適度に発言し、高頻度のスパム投稿で他のメンバーの交流に影響を与えないようにしてください。スパム速度が速すぎる場合は、ランダムにミュート処理されます。

## 四、附則

- 本グループルールは公開日から有効となり、管理グループは実際の状況に応じてルールを修正・補足する権利を有します
- ルールの執行に異議がある場合は、管理者に個別に連絡してフィードバックしてください。グループ内で管理決定を公開質問することは禁止します
